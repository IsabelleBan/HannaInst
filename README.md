# Hanna Instruments – Platform Migration & Custom Development
**Project Type:** Professional E-commerce Migration (LIA)
**Stack:** Shopify (Liquid), Python, REST API
**Status:** Live & Production Ready

## Project Overview
Lead developer for the migration and technical implementation of Hanna Instruments' new e-commerce platform. The project involved a complete transition from PrestaShop to Shopify.

## Key Technical Achievements
* **Data Migration:** Engineered custom Python scripts to extract, transform, and load (ETL) years of legacy data from PrestaShop to Shopify via REST APIs.
* **Custom Liquid Development:** Developed bespoke theme functionality directly in Shopify’s Liquid engine to handle technical product specifications and complex catalogs.
* **API Integration:** Implemented third-party integrations for product management and enhanced search functionality.

## Technical Skills Demonstrated
* **Backend:** Python (Data migration scripts), API Architecture.
* **Frontend:** Liquid, JavaScript, Advanced CSS.
* **Platform:** Shopify Plus / Advanced.

> **Note on Source Code:** As development was performed directly within the Shopify environment, this repository contains the theme files and core architectural components.
